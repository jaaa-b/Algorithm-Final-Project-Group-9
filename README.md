# Dijkstra Algorithm Complexity Analysis with JFreeChart

This project analyzes the time complexity of Dijkstra's algorithm using JFreeChart to visualize the comparison between actual running time and theoretical complexity.
The project requires some technologies.  


   - [jfreechart-1.5.3.jar](https://www.jfree.org/jfreechart/download/)
   - [jcommon-1.0.24.jar](https://www.jfree.org/jcommon/)
